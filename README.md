# The Boilerplate

A boilerplate with a pretentious name. It is composed of:

## Language

- TypeScript

## Server

- Apollo Server
- TypeGraphQL
- Envalid

## Auth

- JWT, extendable
- Passwordless

## Quality

- Jest

## Data

- PostgreSQL
- TypeORM (Active Record)

## Client-side

- Gatsby
- Apollo Client
- Network-Only Service Worker

## Dev

- Docker

## Logging

- Winston
- Sentry, with sourcemaps (3rd-party)
- Loggly (3rd-party)

## Deployment

- Zeit Now (3rd-party)
- Heroku (3rd-party)
- Heroku Postgres (3rd-party)
